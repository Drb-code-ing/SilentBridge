## 演示你自己怎么点（和帖子第 3 节一致）

1. 打开 https://silent-bridge-sigma.vercel.app  
2. 点首页 **医院问诊**  
3. 点 **对方看完了，开始收听**  
4. 点 **演示字幕**（或真麦：开始收听→停止并识别）  
5. 往下看重点/风险  
6. 可 **保存这次重点**，底部进 **记录**
