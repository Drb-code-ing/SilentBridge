# TRAE Work 生成创意方案 HTML 提示词

将下面内容复制到 TRAE Work，用于生成报名所需的创意方案 HTML。

```text
请为 TRAE AI 创造力大赛生成一份可直接作为报名附件的创意方案 HTML 页面。

作品名称：
SilentBridge 无声桥：AI 听障沟通副驾驶

参赛赛道：
主赛道：社会服务赛道
附加赛题：社会公益

一句话介绍：
SilentBridge 无声桥是一款面向听障、弱听、临时失语及不便开口人群的 AI 沟通副驾驶。它在医院问诊、求职面试、课堂会议、政务窗口等关键场景中，将语音对话实时转写成清晰字幕，自动提炼重点，并生成自然、得体、可一键展示给对方的回复建议，让每一句重要的话都不被错过。

页面目标：
让评委在 1 分钟内理解这个作品的社会价值、公益属性、真实需求、AI 创新点和初赛 Demo 可行性。页面应有温度，但不能煽情过度；要体现尊严、平等参与、数字无障碍。

页面结构：
1. 首屏 Hero
   - 标题：SilentBridge 无声桥
   - 副标题：让每一句重要的话，都被清楚听见
   - 简短说明：面向听障、弱听、临时失语及不便开口人群的 AI 沟通副驾驶
   - 视觉建议：安静、清晰、有连接感，可以用字幕流、对话气泡、桥梁或声波可视化作为设计元素

2. 问题背景
   - 说明听障人群在医院问诊、求职面试、课堂会议、政务窗口等关键场景中容易错过重要信息
   - 强调普通字幕只解决“看见文字”，但无法解决“理解重点、确认下一步、从容回应”

3. 目标用户与痛点
   - 听障、弱听人群
   - 临时失语或不便开口的人群
   - 嘈杂环境下难以听清关键信息的人群
   - 医院、学校、政务大厅、招聘面试等公共服务场景
   - 痛点：重要信息错过、临场回复压力、服务方沟通成本、缺少会后摘要

4. 核心功能
   - 场景化实时字幕：医院、面试、课堂、政务窗口
   - 重点提炼卡片：自动整理医嘱、问题意图、课堂重点、办理步骤
   - 一键回复建议：生成自然、得体、可展示给对方的回复
   - 沟通快捷卡：请说慢一点、请写下关键词、我需要确认时间和费用等
   - 会话结束摘要：生成可保存、可分享的对话摘要卡

5. Demo 场景
   - 医院问诊：提炼用药、复诊、注意事项
   - 求职面试：理解问题意图，生成回答提纲
   - 课堂会议：整理重点笔记和待办
   - 政务窗口：转换为材料清单和步骤清单

6. 创新点
   - 从语音转文字升级为沟通理解辅助
   - 聚焦高压力关键场景，而不是泛泛聊天
   - 同时帮助理解和表达，是双向沟通副驾驶
   - 可扩展为公共场所无障碍沟通组件

7. 社会价值
   - 无障碍不是少数人的特殊需求，而是社会基础设施
   - 核心理念：不是让听障者适应嘈杂世界，而是让世界学会把重要的话说清楚

8. 社会公益附加赛题契合点
   - 说明作品计划叠加参与社会公益附加赛题
   - 强调它关注听障、弱听、临时失语及不便开口人群在公共生活中的沟通权利
   - 说明它能降低特殊群体参与社会的隐形门槛
   - 说明它能提升医院、学校、政务大厅、招聘面试等公共服务场景的无障碍沟通能力
   - 强调公益价值不是停留在口号，而是可以被体验、被部署、被复用的 AI 工具

9. 后续计划
   - 初赛完成可交互网页 Demo
   - 保留 TRAE Session ID 和开发过程截图
   - 后续扩展到医院、学校、政务大厅、企业会议等场景

设计风格：
- 温暖、克制、现代、清晰
- 不要可怜化听障用户，要强调尊严、独立和平等参与
- 色彩建议：深墨蓝/暖白作为主背景，使用青绿色或亮蓝色表达连接与科技感
- 页面应适合比赛报名展示，结构清楚，标题醒目
- 生成单文件 HTML，包含内联 CSS，可直接打开预览

请输出完整 HTML 文件。
```
