# 合规检查清单

提交前逐项检查。

## 报名阶段

- [ ] 报名帖标签选择：社会服务。
- [ ] 如果可加附加标签，加入：社会公益。
- [ ] 标题格式符合：报名赛道 + 创意名称。
- [ ] 正文至少包含：创意名称 + 创意介绍。
- [ ] 正文至少包含：目标用户及痛点。
- [ ] 正文至少包含：价值与意义。
- [ ] 已上传 TRAE Work 生成的创意方案 HTML。
- [ ] HTML 文件由 TRAE Work 生成、修改和导出。
- [ ] 已保存 TRAE Work 对话截图。
- [ ] 已保存 TRAE Work 任务历史截图。
- [ ] 已记录 TRAE Work Session ID / 任务 ID。

## 初赛阶段

- [ ] Demo 由 TRAE IDE 创建和修改。
- [ ] 可提供体验链接或 HTML Zip。
- [ ] 至少保存 3 张开发过程截图。
- [ ] 至少记录 3 个关键 Session ID。
- [ ] Demo 能展示核心交互，而不是静态介绍页。
- [ ] 场景切换可用。
- [ ] 字幕模拟可用。
- [ ] 重点提炼可用。
- [ ] 一键回复或快捷沟通卡可用。
- [ ] 摘要生成可用。
- [ ] 桌面端和移动端均检查过。
- [ ] 确认最终作品由 TRAE Work / TRAE IDE 完成生成、修改和导出。

## 执行边界

- TRAE Work：报名创意方案 HTML 的最终生成。
- TRAE IDE：初赛 Demo、代码、样式和交互的最终开发。
